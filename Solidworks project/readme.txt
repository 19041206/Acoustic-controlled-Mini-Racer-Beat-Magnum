Solidworks版本：Solidworks 18   
Car_base文件夹中工程为车子底盘的模型
Transmitter文件夹中工程为遥控器的模型

上述文件夹内包括了装配体模型以及每一个小部件的模型

如果您的Solidworks版本低打不开上述模型，或者没有您安装solidworks，也可以直接使用stl文件进行3D打印
底盘打印时，可以直接打印：
mark3.STL                （1个）
back_lock.STL          （1个）
back_wheel.STL       （2个）
Racer_Lock.STL        （1个）
board_support.STL  （1个）

遥控器打印时，可以直接打印：
Transmitter_base.STL    （1个）
Trans_part2.STL            （1个）
